# Student Grade Calculator

##  Project Overview
This Python program calculates student grades based on marks and provides encouraging feedback messages.

---

##  Features
- Input student name and marks
- Validates marks (0–100 only)
- Uses if-elif-else grading logic
- Provides motivational messages
- Handles invalid input using while loop
---
Grading logic:

| Marks Range | Grade |
|------------|------|
| 90–100     | A    |
| 80–89      | B    |
| 70–79      | C    |
| 60–69      | D    |
| 0–59       | F    |


##  How to Run

```bash
python grade_calculator.py
